# Logo Image

Place your FC25 logo here as `fc25-logo.png`

The logo should be:
- PNG format with transparent background
- Approximately 200-300px height (will be displayed at 40px height)
- Horizontal orientation

Example command to upload from local machine:
```bash
scp fc25-logo.png ubuntu@ec2-3-133-140-158.us-east-2.compute.amazonaws.com:/home/ubuntu/fifa-ranks/backend/app/static/images/
```
